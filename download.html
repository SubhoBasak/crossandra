Installation

Windows

1. Download & extract the zip file.

2. Create a folder named 'Crossandra'. Recommended in C:\ drive.

3. Move the crss.exe file inside the extracted zip folder to the newly created Crossandra folder.

4. Add the newly created Crossandra folder to the environmnet variable.

5. Open up a new powershell or CMD window and type crss.exe. It should show "No input file!". If it is, then congratulation, your installation is successfull.

6. Otherwise check the environmnet variable and restart your system and try agian.

Linux & Mac OS

1. Download & extract the zip file.

2. Open the extracted folder in the terminal.

3. Run the following command the allow the installation script to execute.

chmod 777 install.sh

4. Now run the installation script by entering the following command.

./install.sh

5. Now type 'crss' in the terminal and hit enter. It should show "No input file!". If it is, then congratulation, your installation is successfull.

6. Otherwise restart your system and try agian.

VSCode extension

1. Open VSCode and navigate to extension.

2. Search for "Crossandra".

3. Install the extension with the crossandra logo and developer name "Subho Basak".

Note: This is not the compiler, it's just a syntax highlighter for VSCode. It's an optional step and can be ignored if you are not using VSCode.
